NAME
    fgdiag - Free Geek Hardware Diagnostics
SYNOPSIS
    diag-disk
    diag-modem
    diag-cd
DESCRIPTION
    These commands run diagnostics on a particular type of gizmo and report
    to the database if the gizmo is good or bad.
OPTIONS
    None.
FILES
  ~/.fgrc, /etc/freegeekrc   
    This file contains information on how to report to the database.
    See the example file 'fgrc' for format.
